## Getting Started

### Prerequisites

Update your npm package to the latest version.
```
npm install npm@latest -g
```

### Setup

1. Clone the repository
```
git clone https://github.com/ChiefWoods/quote-api.git
```
2. Install all dependencies
```
npm install
```
3. Start Node.js server
```
npm run start
```
